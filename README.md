# 📊 GlowBox Instagram Ad Spend Optimization

## 🧠 Project Overview
A complete data analysis project solving a real startup problem —
where should GlowBox (a D2C skincare startup) invest its Instagram
ad budget to get the maximum return?

## ❓ Business Problem
Startups waste thousands of rupees running Instagram ads without
knowing which campaigns, audiences, or days perform best.
This project answers 5 key business questions using real data.

## 💡 Key Business Questions Answered
1. Which campaign goal gives the highest ROI?
2. Which target audience converts best?
3. Which day of the week gives the best ROI?
4. Which customer segment is most valuable?
5. Which campaigns are wasting budget?

## 🔍 Key Insights Found
- ✅ **Product Launch** campaigns give the highest ROI (4.02x)
- ✅ **Women 25-34** and **All Ages** are the best performing audiences
- ✅ **Tuesday & Monday** give the best ROI — avoid heavy spend on Sunday
- ✅ **Fashion** segment customers give the highest return
- ✅ Top 10 campaigns all achieved maximum ROI of 8.0x

## 📁 Project Structure

## 📊 Dataset
- Source: Kaggle — Social Media Advertising Dataset
- Total rows: 3,00,000
- Instagram rows used: 75,101
- Columns: 19 features including ROI, CTR, CPA, ROAS

## 📈 KPIs Calculated
- **CTR** (Click Through Rate) — how many people clicked the ad
- **CPA** (Cost Per Acquisition) — cost to get one customer
- **ROAS** (Return on Ad Spend) — return per rupee spent

## 🚀 How to Run
1. Clone this repository
2. Install dependencies: `pip install pandas matplotlib seaborn`
3. Run exploration: `py explore.py`
4. Run cleaning: `py clean.py`
5. Run analysis: `py analysis.py`
6. Run charts: `py charts.py`
